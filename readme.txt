== Description ==

Single Image LightBox plugin was created to add lightbox option to the single image in the post or page.

Features of this plugin:

* Adds lightbox feature to the already uploaded images.
* Avoid opening images in the other tab or window.

== Installation ==

1. Upload the Single Image LightBox plugin folder to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress.

3. Done!
